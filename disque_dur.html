Le disque dur

Le disque dur est un composant de stockage indispensable. Il permet de conserver toutes les données de l’ordinateur même lorsqu’il est éteint : système d’exploitation, logiciels, documents, photos et vidéos.

Fonctions principales

  • Stocker les données de façon permanente
  • Conserver le système d’exploitation
  • Enregistrer les fichiers personnels

Comparaison des types de disques

Type Vitesse Capacité Prix
HDD Lente Élevée Moins cher
SSD Très rapide Moyenne Plus cher

À retenir :
Le disque dur permet de stocker les données sur le long terme. Les SSD sont plus rapides que les HDD et améliorent fortement les performances.
